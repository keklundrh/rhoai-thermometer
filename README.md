# RHOAI Thermometer

This is an analytics tool requiring manual steps before running the first time: 
1. in project root, `git clone https://github.com/red-hat-data-services/rhoai-disconnected-install-helper.git`
2. install
  * grype 
  * syft
  * jq 
  * git
